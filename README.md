## 🎮 Game Overview
A simple, classic Pong game made in Unity using C#.  
Includes:
- Basic ball and paddle physics  
- AI opponent with customizable difficulty  
- Score system and UI

## 📦 Build Info
- **Engine**: Unity 6.1 (WebGL compatible)
- **Version**: V1.0
- **Platform**: Web
